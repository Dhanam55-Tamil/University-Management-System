# Government Architecture Science College Website

## Project Description

This project is a simple and responsive college website created using **HTML and CSS**.
The website provides basic information about the college such as the home page, about section, departments, and campus details.

## Features

* Responsive navigation bar
* College logo and title
* Background image for the home page
* Welcome section
* Explore / Our Campus button with smooth scrolling
* Clean and simple user interface

## Technologies Used

* HTML5
* CSS3

## Project Structure

* `index.html` – Main homepage of the website
* `style.css` – Styles and design of the website
* `images/` – Contains logo and background images

## Purpose of the Project

The purpose of this project is to design a basic college website interface and practice front-end
